# Crypto Price Tracker

This project tracks real-time prices of Bitcoin, Ethereum, and Dogecoin using the CoinGecko API.

## Features
- Real-time price tracking (update every 60 seconds)
- Saves historical prices in CSV file
- Simple terminal display

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the project:
   python main.py
